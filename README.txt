== Description ==

畅言是一个简单友好的社会化评论及聚合平台。用户可以使用社交账号一键评论同步至社交网络，将内容和自己的评论分享给好友。增加第三方网站用户活跃度，调动好友参与评论，帮助网站实现社会化媒体优化SMO，有效提升网站社会化流量！

= 引入社交平台的人气与流量 =

* 社会化账号登录。无需注册，用户直接使用新浪微博、qq等社交账号登录，降低登录门槛，增加评论人气；
* 分享到社交平台。用户发评论时，评论可直接分享到各大社交平台上，并可自动@您的官方微博；
* 评论回流。分享出去的评论在社交平台有了回复，这些回复会被引入到畅言中；您网站的官方微博收到的回复，也会被引入到畅言中；
* @好友（即将推出）。看到好玩的文章或评论，用户可以@微博或qq的好友，将好友带入到讨论中。

= 显著提升网站的评论量 =

* 评论联通。首创评论连通功能，打通相关网页，汇聚全网优质评论；
* 盖楼。回复即盖楼，鼓励互动，让评论更有趣；
* 跨网站回复提醒。访问任意使用畅言的站点，用户都能第一时间收到回复提醒；
* 热门评论。热门评论置顶展示，用户能快速浏览到优质评论；
* 评论PK（即将推出）。发起辩题，吸引用户表达态度以及为支持的观点辩护，形成正反方大讨论。

= 三重过滤系统+浏览器管理插件，过滤99.99%的垃圾评论 =

* 机器智能过滤。第一重过滤：机器智能过滤，过滤掉广告、色情、灌水等；
* 畅言人工审核。第二重过滤：根据国家政策法规，实时更新违禁词库，灵活掌控审核尺度；
* 站长审核平台。第三重过滤：管理本站所有用户与评论，拥有自定义违禁词、关闭单篇评论等多种审核工具，使您的审核更加简单高效；
* 浏览器管理插件。首创浏览器管理插件，可在网页评论列表直接管理评论。

= 与您的网站完美融合 =

* 换肤。多款皮肤，适合不同风格的网站；
* 个性化配置。个性化设置，提供多种功能、样式供选择；
* 单点登录。深度打通本站账号，既可使用本站账号，又能享受社交账号的便利；
* 评论导入导出。一键导入历史评论数据，一键导出畅言评论数据。

= 全面支持移动端 =

* 支持wap、APP接入，满足各种移动端接入；；
* 支持文字、图片评论，后续还将提供移动定位、甚至语音等富媒体评论支持，充分发挥移动端优势；
* 如果您的网站含有多种终端，使用畅言将使得您的PC端与移动端保持一致，提升用户体验；

1. 在WordPress 插件库中搜索"changyan", 下载并启用；
2. 在WordPress “设置 -> 畅言评论”中，注册并绑定账号；
3. 同步本地评论到畅言，并进行其他设置。

== Frequently Asked Questions ==

= 畅言是什么？ =

畅言是业内领先的社会化评论系统，支持PC和移动端两种接入方式，为网站提供了一种全新的评论方式。畅言拥有多种常用账号登录、三重审核机制、实时的数据统计、快捷数据导出、全面支持移动端等领先功能。充分满足了当前各大网站对于用户登录、评论、分享 、审核等方面的需求。

= 畅言如何保证数据安全？ =

网站接入畅言后，虽然评论数据产生在畅言服务器，但后台提供快捷数据导出与开放api接口等方式保障网站实时获取本站数据。同时，畅言在网站需要时，可以签订保密协议，保障所有数据的安全性。

= 畅言如何保证审核安全？ =

畅言在业内率先实现三重审核机制。当评论数据产生后，首先经过搜狐垃圾过滤系统，防灌水、防垃圾、防广告；然后经过搜狐专业的人工审核团队，严格清理政治敏感性信息，同时违禁词库会根据国家政策通知，实时更新；最后是网站的自主审核，所有畅言合作网站都有专属的管理后台，管理所有本网评论，即使经过搜狐审核的评论，网站也有权限再次审核。网站对本站评论拥有完全的管理，可以设置先发后审与先审后发等多种审核形式。

= 如何保证畅言服务的持续性，可靠性？ =

畅言评论，作为搜狐网评论系统的升级版，本身就在为搜狐服务，产品的安全与稳定性毋庸置疑。同时，畅言可以与合作网站签订安全协议，保障合作网站安全稳定性需求。

= 畅言是免费的吗？ =

畅言作为新一代的评论系统，免费为广大合作网站提供安全可靠的服务。但畅言保留可能的增值服务收费权利。

= 网站如何获取评论数据？ =

所有合作网站，在畅言管理后台，都有一键数据导出功能，网站可以根据需要选择导出数据。同时，畅言提供开放接口，满足网站更多元化数据导出需求。另外，对于网站之前的评论数据也可以通过畅言导入，评论永不丢失。

= 网站账号如何与畅言结合？ =

目前畅言提供单点登录方案能够解决此问题，具体请参见单点登录。

= 什么是畅言vip账号，vip账号如何申请？ =

VIP账号，拥有分频道管理与统计分析、管理员体系设置、一对一技术客服等多种功能与服务，详情请联系畅言客服。

= 畅言是否支持多频道，多账号管理？ =

通过申请畅言vip账号，获取多频道，多账号管理服务，如有需要请联系客服。

= 如何关闭评论？ =

畅言支持关闭单篇文章评论的功能。
