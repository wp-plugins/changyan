== Description ==

畅言是一个简单友好的社会化评论及聚合平台。用户可以使用社交账号一键评论同步至社交网络，将内容和自己的评论分享给好友。增加第三方网站用户活跃度，调动好友参与评论，帮助网站实现社会化媒体优化SMO，有效提升网站社会化流量！

= 一键评论 实时同步 =

* 畅言社会化评论系统是国内领先的第三方实时社交评论工具，“完全社交化”可将评论一键同步到各大微博与社区（新浪微博、QQ、人人、搜狐）；
* 通过畅言发出的评论同步到微博等社会化媒体，提高用户评论积极性，帮助网站建立活跃社交评论社区，进而提升网站评论质量，扩大传播效果；
* 畅言实时抓取社交网络评论，将评论的回复与跟帖同步至使用的网站上，使网站用户之间进行更加深入的交互。增加了网站社会化属性，是连接网站与社交网络及其他网站的全网络社交评论工具。

= 垃圾评论屏蔽 =

* 畅言主动拦截屏蔽各种垃圾评论、恶意评论，营造网站良好的互动氛围；

== Frequently Asked Questions ==

= 畅言评论系统是免费的吗？ =

是的，免费使用专业的系统服务，心动不如行动，快快体验吧。

= 畅言能给网站带来什么？ =

更多用户：支持新浪微博、QQ等社交网站用户登录，获取用户社交网络信息。帮助网站实现用户转化 。
更多流量：支持一键同步及分享，获得更多推广和流量。微博及社交网络评论回流，获得更多评论。 
更高品质：稳定可靠的评论服务。免费云过滤系统，智能处理垃圾评论，净化评论氛围。 
更好管理：无需任何开发，全面支持web、wap及客户端应用。支持站长个性化设置、审核服务。免费的评论数据分析服务。

= 畅言会不会影响网站速度？ =

畅言社会化评论框是通过异步加载的方式来获取数据，不会影响到页面其它内容的加载速度。

= 畅言用户名、密码和注册邮箱安装后可以更改吗？ =

畅言用户名和注册邮箱一但注册成功是无法更改的。如果您需要变更账号和用户名，可以重新注册一个畅言账号。如果想更改密码，可以通过畅言管理后台中的账号管理来更改密码。

= 在后台网站列表处，不小心删除了网站地址，该如何处理？ =

把网站现有的畅言代码删除掉，清理网站缓存后，重新将代码嵌入到网站，服务器缓存5分钟左右，在后台即可恢复网站列表。

= 如果长期不登录畅言，评论数据会不会被删除？ =

所有注册畅言并已产生社会化评论的网站，其评论数据会永久保存。而且数据是保存在畅言服务器中，不会占用网站服务器空间。

= 畅言有哪些审核方式？ =

畅言目前提供两种评论审核方式，一种是先发后审，即对评论只进行事后审核，即发即显；另一种是先审后发，即对评论进行事前审核，所有评论需要经过审核才能出现在相应页面。站长可以在「评论设置」中的「审核设置」进行评论审核方式的选择。

= 畅言如何审核网站的评论？ =

畅言审核系统内的评论分为3类，待审核评论、已审核评论、已删除评论。首先，用户发表的评论将会经过畅言的垃圾信息过滤系统。其次对于先审后发的评论会先进入待审核评论中，站长可以通过或删除评论。通过的评论将在显示在已通过评论列表中，删除的评论将显示在已删除评论列表中。另外如果用户发表的评论包含站点设置的非法词汇或者用户被站点加入黑名单，评论将直接被删除，进入已删除评论列表；用户发表的评论如果包含站点设置的敏感词汇，评论将直接进入待审核列表，敏感词汇会做标红处理。

= 什么是非法词库和敏感词库？ =

非法词库和敏感词库是畅言提供的专业评论内容过滤服务，站点可以根据自己的需求，自行添加，删除这两个词库。 
非法词是指违反相关法律法规的词汇，如果用户发表的评论包含任何非法词汇（精确匹配），则该条评论将被自动删除，进入已删除列表并做标红处理。 
敏感词是指可能涉及敏感内容的词汇，如果用户发表的评论包含任何敏感词汇（精确匹配），则无论审核策略是否先发后审还是先审后发，该条评论都将直接进入待审核列表并作标红处理。

= 什么是黑白名单？ =

畅言的黑白名单功能是对黑名单，白名单以及IP封禁功能的总称。站点可以对具有扰乱站点评论秩序，发表恶意评论等不良行为的用户进行拉黑，黑名单上的用户发表的评论将直接进入已删除评论列表。站点可以将具有特殊需求的用户添加至白名单，白名单上的用户发表的评论将直接进入已通过评论列表。站点可以将具有危害的IP添加至IP封禁名单，IP封禁名单上的用户发表的评论将直接进入已删除评论列表。
